# 🧠 Text Summarizer (NLP Project)

This project is a Transformer-based Text Summarization system built using T5.

## 🚀 Features

* Summarizes long text into short meaningful summaries
* Built using HuggingFace Transformers
* Simple UI using Gradio
* Real-time text input support

## 🛠️ Tech Stack

* Python
* Transformers (T5)
* PyTorch
* Gradio

## 📂 Project Structure

app.py
requirements.txt
text_summarizer.ipynb

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python app.py
```
