# 🧠 Text Summarizer (Work in Progress)

This project is a Transformer-based Text Summarization system built using T5.
It is currently under development and being actively improved.

## 🚀 About the Project

The goal of this project is to build a system that can generate meaningful summaries from dialogue-based text.
This is one of my major NLP projects, and I am continuously working on improving its performance and usability.

## ⚙️ Current Status

* Model is trained on dialogue summarization dataset (Samsum)
* Basic summarization pipeline is implemented
* Gradio-based interface is created
* Output quality is still being improved

## 🛠️ Tech Stack

* Python
* Hugging Face Transformers (T5)
* PyTorch
* Gradio

## 📂 Project Structure

* app.py → Gradio app
* requirements.txt → dependencies
* text_summarizer.ipynb → training & experimentation

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python app.py
```
